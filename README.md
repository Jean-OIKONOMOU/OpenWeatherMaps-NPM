# HOW DO I INSTALL THIS AMAZING NPM MODULE ?
```console
npm install -g jadm
```

# HOW DO I CALL THIS AMAZING MODULE ?
In the console type **jadm** followed by your city.
For example:
```console
jadm brussels
```
